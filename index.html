GIT

Listado con comandos para utilizar GIT:

  1. Descargar desde el sitio oficial haciendo click aqui
  2. Abrir GIT BASH o la Terminal de nuestro sistema.
  3. Configuraciones por única vez:
      Email: git config --global user.email "micorreo@gmail.com"
      Nombre: git config --global user.name "Juán Pérez"
  4. Localizar la carpeta de nuestro proyecto.
  5. git init para empezar a trabajar con git en nuestro proyecto.
  6. git add . para agregar todos los archivos que contenga la carpeta o git add + el nombre del archivo que queremos agregar. Ejemplo: git add index.html
  7. git commit -m “Mensaje” Mensaje obligatorio para indicar modificaciones realizadas.
  8. git log --oneline Para conocer los códigos de los commits realizados.
  9. git branch nombre_rama para crear una nueva rama.
  10. git branch -D nombre_rama para eliminar dicha rama.
  11. git checkout nombre_rama para moverse a una rama.
  12. git merge nombre_rama debemos estar en MASTER para fusionar los cambios de distintas ramas.
  13. git status para ver en que estado estamos trabajando.
